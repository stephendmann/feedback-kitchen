📖 What is a JSON file and why would I use one?

A .json file is simply a portable save file for a scorer. It's a small text file that stores all the rubric settings — criteria, grade bands, feedback templates — in a format Feedback Kitchen can read back exactly.

When would you use this?

  • A colleague built a scorer and emailed or shared the .json file with you
  • You exported your own scorer from another computer or browser and want to restore it here
  • Your department is sharing a standard rubric across multiple markers

Nothing is uploaded to any server — the file is read directly in your browser and saved only on this device.
