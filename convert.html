Convert Assessment Manual

Upload a PDF assessment manual and Feedback Kitchen will detect the assessment structure, extract the rubric, and generate a ready-to-use scorer JSON.

How it works

  1. Upload your assessment manual PDF (up to 6 MB)
  2. The tool scans the document and lists all detected assessment sections
  3. Select one assessment — the rubric, criteria, and weights are extracted
  4. Review and edit the result, then download the FK scorer JSON

The PDF is sent to a secure server for AI processing and is not stored. Only PDF files are supported in this version.
